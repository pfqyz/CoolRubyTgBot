# Марков Бот

Telegram-бот для работы с алгоритмами Маркова. Позволяет вводить систему правил подстановки и применять их к заданному слову.

## Команды

| Команда | Описание |
|---------|----------|
| /start | Запустить бота и показать главное меню |
| /help | Показать справку |
| /stop | Завершить сессию |

## Возможности

- Ввод системы правил в формате A->B или A->.B (точка означает завершающее правило)
- Ввод исходного слова
- Автоматическое применение алгоритма Маркова
- Удаление последнего правила
- Очистка всей системы
- Интерактивные кнопки для удобного управления

## Инструкция по использованию

### Главное меню

После запуска вы увидите клавиатуру с кнопками:

| Кнопка | Что делает |
|--------|-------------|
| Ввести систему | Начать ввод правил алгоритма Маркова |
| Ввести слово | Ввести слово, которое нужно обработать |
| /help | Показать справку |
| /stop | Завершить работу с ботом |

### Как ввести систему правил

**Шаг 1:** Нажмите кнопку "Ввести систему"

**Шаг 2:** Вводите правила по одному в формате:

- `a->b` - заменить букву a на b
- `ab->c` - заменить последовательность ab на c
- `a->.b` - завершающее правило (после его применения алгоритм остановится)

**Шаг 3:** После каждого правила бот показывает текущую систему

**Шаг 4:** Когда все правила введены, нажмите "Завершить ввод"

**Также вы можете воспользоваться доп функциями при вводе системы:**
- "Удалить последнее правило" - удаляет только что добавленное правило
- "Очистить систему" - удаляет ВСЕ правила
- "Главное меню" - отменить ввод и вернуться в меню

### Как ввести слово

**Шаг 1:** Нажмите кнопку "Ввести слово"

**Шаг 2:** Введите любое слово (например, abab)

**Шаг 3:** Нажмите "Завершить ввод"

### Результат:

После того как введены и система правил, и слово:

**Шаг 2:** Бот покажет:
- Введённую систему правил
- Исходное слово
- Результат применения алгоритма

## Завершение работы

Чтобы закончить работу с ботом:
- Нажмите кнопку "/stop" или отправьте команду `/stop`
- Бот попрощается и удалит клавиатуру
